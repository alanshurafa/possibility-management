# Possibility Management

Materials and tools for [Possibility Management](https://possibilitymanagement.org/) — a body of research and practice for personal and cultural transformation, contexted in radical responsibility.

## What's here

This repo is the public home for Possibility Management-related work: documents, distinctions, thoughtware experiments, and supporting tools.

Related: [ETB-Expand-The-Box](https://github.com/alanshurafa/ETB-Expand-The-Box) — digital course versions of Expand The Box.

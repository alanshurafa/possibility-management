The practice
I.
Guided sequence · ~15 min · solo

Centering Practice.

Find your center. Drop a grounding cord. Set a bubble. Open the golden cube. Close cleanly. Note what shifted.
The canonical PM four-step energetic-body sequence — guided, paced, with body diagrams that build through the practice. A breath rhythm of ten in / ten out runs alongside each step. At the end, a short journal capture saves on this device, so you can see what landed last time when you come back tomorrow to do it again. By the third or fourth pass the equipment is yours.

How to use this.

Read the Day 3 module first (`Days/Day 03 - Liquid State, Center-Ground-Bubble, Five Bodies.md`). Watch the two source videos. Then come here.

Run the practice once today. Sleep on it. Run it again tomorrow. Run it a third time the day after that. The energetic body's perceptual interface comes online by being practised — not by being insisted on.

Then run the partner exchange in the module, with the equipment now installed rather than theoretical.

What the tool saves.

Your practice note from each pass is stored in your browser's local storage — on this device only, never sent anywhere. The next visit shows the date, rep count, and the last note you wrote.

Clear your browser data and the tool forgets. That is the design. This is your practice, not the course's record.

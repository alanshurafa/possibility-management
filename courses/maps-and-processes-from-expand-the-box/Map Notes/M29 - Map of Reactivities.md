# M29 — Map of Reactivities

*Six different kinds of automatic reaction — Buttons, Voices, Activated Traumas, Hooks, Triggers, Imbalances — that all feel the same from inside but each route to a different tool, so the first move is always to name which one fired.*

![Map of Reactivities — Possibility Management map](../Maps/M29.png)

**What it is.** A reactivity is when something external sets off an automatic internal reaction that is *not* a present-time conscious choice — you are already reacting before you decided to. The load-bearing claim: not all reactivity is the same kind, and each kind has a different antidote. From the inside they all feel identical — a jolt, a charge, a sudden *no* — so the untrained move is to reach for one favorite tool every time. The map says stop and ask *which of the six is this?*, because a Button needs Box work, an Activated Trauma the EHP, and an Imbalance just food, rest, or grounding. Misdiagnosing the kind applies the wrong tool, and the wrong tool does nothing.

**At a glance.** Buttons → Box rules firing → Expand the Box (M04) · Voices → swallowed authority figures → shoot the voices · Activated Traumas → the past flooding the present → EHP (M12) for the in-scope stored emotion; actual trauma goes to a clinician, not the EHP · Hooks → gremlin seductions into low drama → decline the gremlin food · Triggers → specific draining sentences / "vampires" → close the cake · Imbalances → a tired/hungry/leaking 5-body (M06), not psychological → bring the body to balance. **Diagnostic move:** name the kind first; the name tells you the route.

## How to explain it verbally

A reactivity is when something outside sets off an automatic internal reaction that is not a present-time choice; you react before you decided to. The load-bearing claim is that not all reactivity is the same kind, and each kind routes to a different tool. From the inside they feel identical, just a jolt, a charge, a sudden no, so the untrained move is to reach for one favorite tool every time. The map names six. A Button is a Box rule firing, so you do Box work. An Activated Trauma goes to the EHP if in-scope, or to a clinician if real. An Imbalance just needs food, rest, or grounding.

**If you only remember one thing:** they all feel the same from inside, so name which of the six fired before you reach for a tool.
